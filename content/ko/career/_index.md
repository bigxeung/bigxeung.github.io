## <span class="section-title">💡 SKILLS & GOALS</span>



### 주요 기술 (Technical Skills)

- **Languages**: Python (익숙), C/C++ (수업 경험), Java (기초)

- **ML & Data Science**: Scikit-learn, Pandas, NumPy, Matplotlib, PyTorch (기초)

- **Tools**: Git, GitHub, Jupyter Notebook, Linux (기초)



### 학습 목표 (Learning Goals)

- **다양한 머신러닝 모델 경험하기**: 회귀, 분류 외에 클러스터링, 추천 시스템 등 다른 종류의 모델도 다뤄보고 싶습니다.

- **데이터 분석 능력 향상**: 더 복잡한 데이터를 다루며 의미 있는 정보를 찾아내는 전처리 및 시각화 능력을 키우고 싶습니다.

- **웹 개발 기초 학습**: 간단한 웹 프레임워크(Flask 등)를 배워, 직접 만든 모델을 눈으로 확인할 수 있는 결과물로 만들어보고 싶습니다.



### 개발자로서의 강점 (Developer Qualities)

- **끈기 있는 문제 해결**: 어려운 개념이나 에러를 마주쳤을 때, 포기하지 않고 원인을 찾아 해결하는 것을 중요하게 생각합니다.

- **협업과 소통**: Git을 사용한 버전 관리와 이슈 트래킹에 익숙하며, 팀원의 의견을 경청하고 적극적으로 소통합니다.

- **새로운 지식에 대한 호기심**: 인공지능 분야의 새로운 소식에 꾸준히 관심을 가지며, 스터디를 통해 지식을 넓히는 것을 즐깁니다.
